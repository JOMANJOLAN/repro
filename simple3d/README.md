### 🎮 SIMPLE3D - 基础3D立方体渲染器
* **日期**：2026/01/23
* **使用语言**：HTML、JavaScript、Rust
* **开发工具**：Firefox
  ```shell
  firefox index.html
  ```
* **项目描述**：
  * 使用纯JavaScript和HTML5 Canvas实现的基础3D图形渲染，完整展示了从3D空间到2D屏幕的渲染
  * 使用Rust标准库实现基础3D图形渲染
* **核心功能**：
  * **web**：
    * **线框渲染**：显示立方体边线与顶点
    * **透视投影**：模拟人眼观察的3D视觉效果
    * **动态旋转**：支持绕Y轴连续旋转动画
    * **坐标变换**：实现完整的3D图形渲染管线
  * **rust**：
    * **canvas**：实现直线的裁剪
    * **coordsys**：简单三维坐标系
    * **object**：三维物体的数据和运算
    * **project**：三维坐标转换二维归一化坐标，二维坐标转换屏幕坐标
    * **transform**：变换矩阵运算
    * **vector**：向量运算
* **学习重点**：
  * 3D坐标系与变换矩阵
  * 模型-视图-投影变换链
  * web：
    * 简单动画循环实现
    * Canvas 2D绘图API使用
* **关键技术实现**：
  * 透视投影（基于`z`坐标的除法运算）
  * 物体平移、缩放（变换矩阵运算）
  * 绕轴3D旋转（直接运算和变换矩阵运算）
  * 从归一化`[-1, 1]`坐标到屏幕坐标的映射
  * web：
    * 双动画循环支持（`setTimeout`与`requestAnimationFrame`）
